Export & Import Data

Download All Chef Data

Downloads all chef profiles from Supabase as a JSON backup file.

Import Chefs from JSON

Upload a JSON file to bulk-add chef profiles to Supabase. Accepts an exported backup file or a simple list of chefs. Each chef's availability, cuisines, dishes and other details are imported automatically.

Cuisine, work type, restaurants, gallery and dishes may be arrays or comma-separated text. Existing profiles are never changed — new ones are added (and start unapproved unless the file marks them approved).
